# Описание использования программы:

Поддерживаются запросы вида:

```
word1 & word2 | word3		
word1 & ~word2 | ~word3
( word1 | word2 ) & word3
( word1 | ~word2 ) & ( word3 | word4 )
```

- `&` - операция AND
- `|` - операция OR
- `~` - операция NOT 


**Результат:
Номера страниц, подходящие под запрос**

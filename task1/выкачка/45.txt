<p><b>Каштановоголовый пушистый погоныш</b><sup class="reference" id="cite_ref-beme_1-0"><a href="#cite_note-beme-1"><span class="cite-bracket">[</span>1<span class="cite-bracket">]</span></a></sup> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Sarothrura lugens</span>) — вид <a href="/wiki/%D0%9F%D1%82%D0%B8%D1%86%D1%8B" title="Птицы">птиц</a> семейства <a href="/wiki/Sarothruridae" title="Sarothruridae">Sarothruridae</a> отряда <a href="/wiki/%D0%96%D1%83%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BE%D0%B1%D1%80%D0%B0%D0%B7%D0%BD%D1%8B%D0%B5" title="Журавлеобразные">журавлеобразных</a>. Птица среднего размера с пёстрым чёрно-белым <a class="mw-redirect" href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86" title="Оперение птиц">оперением</a> тела, <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D1%8B%D0%BB%D0%BE_%D0%BF%D1%82%D0%B8%D1%86" title="Крыло птиц">крыльев</a> и <a href="/wiki/%D0%A5%D0%B2%D0%BE%D1%81%D1%82" title="Хвост">хвоста</a>, каштановой головой и белым горлом. Обитает на полях и болотах, покрытых густой высокой травой, на лоскутном <a href="/wiki/%D0%90%D1%80%D0%B5%D0%B0%D0%BB" title="Ареал">ареале</a> в центральной части <a class="mw-redirect" href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0_%D1%8E%D0%B6%D0%BD%D0%B5%D0%B5_%D0%A1%D0%B0%D1%85%D0%B0%D1%80%D1%8B" title="Африка южнее Сахары">Африки южнее Сахары</a>. В желудках птиц были обнаружены <a href="/wiki/%D0%9D%D0%B0%D1%81%D0%B5%D0%BA%D0%BE%D0%BC%D1%8B%D0%B5" title="Насекомые">насекомые</a> и семена. <a href="/wiki/%D0%93%D0%BD%D0%B5%D0%B7%D0%B4%D0%BE_%D0%BF%D1%82%D0%B8%D1%86%D1%8B" title="Гнездо птицы">Гнёзда</a> и <a href="/wiki/%D0%AF%D0%B9%D1%86%D0%BE" title="Яйцо">яйца</a> описаны не были, единственный известный экземпляр птенца был покрыт чёрным пухом. <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7_%D0%BE%D1%85%D1%80%D0%B0%D0%BD%D1%8B_%D0%BF%D1%80%D0%B8%D1%80%D0%BE%D0%B4%D1%8B" title="Международный союз охраны природы">Международный союз охраны природы</a> относит его к <a href="/wiki/%D0%92%D0%B8%D0%B4%D1%8B,_%D0%B1%D0%BB%D0%B8%D0%B7%D0%BA%D0%B8%D0%B5_%D0%BA_%D1%83%D1%8F%D0%B7%D0%B2%D0%B8%D0%BC%D0%BE%D0%BC%D1%83_%D0%BF%D0%BE%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D1%8E" title="Виды, близкие к уязвимому положению">видам, близким к уязвимому положению</a>.
</p><p>Каштановоголовый пушистый погоныш был описан <a href="/wiki/%D0%91%D1%91%D0%BC,_%D0%A0%D0%B8%D1%85%D0%B0%D1%80%D0%B4" title="Бём, Рихард">Рихардом Бёмом</a> в 1884 году. <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7_%D0%BE%D1%80%D0%BD%D0%B8%D1%82%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B2" title="Международный союз орнитологов">Международный союз орнитологов</a> выделяет два <a href="/wiki/%D0%9F%D0%BE%D0%B4%D0%B2%D0%B8%D0%B4" title="Подвид">подвида</a>. В роду <a href="/wiki/%D0%9F%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B5_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88%D0%B8" title="Пушистые погоныши">пушистых погонышей</a> он образует <a href="/wiki/%D0%A1%D0%B5%D1%81%D1%82%D1%80%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0" title="Сестринская группа">сестринскую группу</a> с <a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Африканский пушистый погоныш">африканским пушистым погонышем</a>.
</p>

<div class="mw-heading mw-heading2"><h2 id="Описание"><span id=".D0.9E.D0.BF.D0.B8.D1.81.D0.B0.D0.BD.D0.B8.D0.B5"></span>Описание</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Описание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;action=edit&amp;section=1" title="Редактировать код раздела «Описание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Каштановоголовый пушистый погоныш — птица среднего размера с длиной тела около 15<span class="nowrap"> см</span>. Длина <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D1%8B%D0%BB%D0%BE_%D0%BF%D1%82%D0%B8%D1%86" title="Крыло птиц">крыла</a> номинативного подвида <i>Sarothrura lugens lugens</i> составляет <span class="nowrap">75—82 мм</span>, подвида <i>Sarothrura lugens lynesi</i> — <span class="nowrap">70—76 мм</span><sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-0"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup>; длина <a href="/wiki/%D0%9A%D0%BB%D1%8E%D0%B2" title="Клюв">клюва</a> — <span class="nowrap">13—15 мм</span>, <a href="/wiki/%D0%A6%D0%B5%D0%B2%D0%BA%D0%B0" title="Цевка">цевки</a> — <span class="nowrap">19—22,5 мм</span><sup class="reference" id="cite_ref-taylor_3-0"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>. Разница в размерах — это единственное различие между подвидами<sup class="reference" id="cite_ref-_d2302afc0ad7fbea_4-0"><a href="#cite_note-_d2302afc0ad7fbea-4"><span class="cite-bracket">[</span>4<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-roux_5-0"><a href="#cite_note-roux-5"><span class="cite-bracket">[</span>5<span class="cite-bracket">]</span></a></sup>.
</p><p><a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D0%B5_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Оперение (биология)">Оперение</a> самцов на голове, затылке и бровях насыщенного каштанового цвета; подбородок и горло белые. Довольно длинный <a href="/wiki/%D0%A5%D0%B2%D0%BE%D1%81%D1%82" title="Хвост">хвост</a> каштановоголового пушистого погоныша окрашен в чёрный цвет с белыми пятнами; птицы часто задирают его вверх. Оперение груди пёстрое, с большим количеством белых полос, которые становятся шире ниже по телу. Оперение сверху тёмное коричнево-чёрное с белыми полосами. Крылья в основном покрыты пятнами или поперечными полосами. В отличие от самцов с головой, окрашенной в однотонный каштановый цвет, в оперении головы самок присутствуют светло-каштановые и чёрные полосы. Белые полосы на спине у самок шире и короче, чем у самцов. Вдоль переднего края крыла и у самцов и у самок проходит тонкая белая полоса. <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BA%D0%B0" title="Радужка">Радужка</a> глаза каштановоголового пушистого погоныша тёмная серовато-коричневая. Клюв и ноги окрашены в коричневый или тёмно-серый цвета. Молодые самцы по сравнению со взрослыми имеют блёклое оперение с тёмной черновато-коричневой головой и меньшим количеством полос. Молодые самки также блёклые, со светлыми, а не каштановыми, полосами на голове; по сравнению со взрослыми птицами у них меньше отметин в верхней части тела и больше — в нижней. Птенцы имеют черноватое оперение с довольно заметным белым пятном на подбородке и горле, а также по центру живота<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-1"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-taylor_3-1"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>.
</p><p>Учёные сравнивают внешний вид каштановоголового пушистого погоныша с <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D0%BE%D0%B3%D1%80%D1%83%D0%B4%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Красногрудый пушистый погоныш">красногрудым</a> (<i>Sarothrura rufa</i>) и <a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Африканский пушистый погоныш">африканским</a> (<i>Sarothrura boehmi</i>) пушистыми погонышами. Этот вид отличается от них более длинным хвостом и более насыщенным каштановым цветом оперения. По сравнению с красногрудым пушистым погонышем оперение темнее, на груди у самца белые полосы, а не пятна, в верхней части тела самки больше отметин, а клюв красногрудого пушистого погоныша полностью чёрный. По сравнению с африканским пушистым погонышем у самца каштановый цвет оперения головы не переходит на затылок, по сторонам шеи и до верхней части груди, у самки пятна в нижней части белые, а не светлые<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-2"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-taylor_3-2"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>. Вместе с тем, в <a href="/wiki/%D0%97%D0%B8%D0%BC%D0%B1%D0%B0%D0%B1%D0%B2%D0%B5" title="Зимбабве">Зимбабве</a> молодых самок <a href="/wiki/%D0%9F%D1%91%D1%81%D1%82%D1%80%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Пёстрый пушистый погоныш">пёстрого пушистого погоныша</a> (<i>Sarothrura elegans</i>) часто некорректно определяют как самок каштановоголового пушистого погоныша<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-3"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup>.
</p><p><a href="/wiki/%D0%92%D0%BE%D0%BA%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F_%D0%BF%D1%82%D0%B8%D1%86" title="Вокализация птиц">Вокализация</a> этого вида пушистых погонышей изучена слабо. Песня самца напоминает непрерывную серию гортанных стонов и уханий «whooo», каждый сигнал которой продолжается около одной секунды с разной скоростью и высотой звука<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-4"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-taylor_3-3"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>. Пауза между такими сигналами может достигать одной минуты. Обычно песни исполняются во время сезона размножения, но в <a href="/wiki/%D0%93%D0%B0%D0%B1%D0%BE%D0%BD" title="Габон">Габоне</a> их отмечали круглый год, особенно в октябре или во время дождя<sup class="reference" id="cite_ref-taylor_3-4"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>. Территориальные позывки представляют собой серию громких и коротких сигналов «koh», издаваемых со скоростью до 3 сигналов в секунду на протяжении 30—45 секунд. Иногда птицы исполняют несинхронные дуэты. На воспроизведение записанных позывок могут отвечать низким сигналом «cuk», схожим с позывкой африканского пушистого погоныша<sup class="reference" id="cite_ref-taylor_3-5"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-_b96edb379d04b541_6-0"><a href="#cite_note-_b96edb379d04b541-6"><span class="cite-bracket">[</span>6<span class="cite-bracket">]</span></a></sup>.
</p>

<div class="mw-heading mw-heading3"><h3 id="Ареал_и_среда_обитания"><span id=".D0.90.D1.80.D0.B5.D0.B0.D0.BB_.D0.B8_.D1.81.D1.80.D0.B5.D0.B4.D0.B0_.D0.BE.D0.B1.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D1.8F"></span>Ареал и среда обитания</h3><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Ареал и среда обитания»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;action=edit&amp;section=3" title="Редактировать код раздела «Ареал и среда обитания»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Каштановоголовый пушистый погоныш обитает на территории таких стран как <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BE%D0%BB%D0%B0" title="Ангола">Ангола</a>, <a href="/wiki/%D0%9A%D0%B0%D0%BC%D0%B5%D1%80%D1%83%D0%BD" title="Камерун">Камерун</a>, <a href="/wiki/%D0%94%D0%B5%D0%BC%D0%BE%D0%BA%D1%80%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0_%D0%9A%D0%BE%D0%BD%D0%B3%D0%BE" title="Демократическая Республика Конго">Демократическая Республика Конго</a>, <a href="/wiki/%D0%A0%D1%83%D0%B0%D0%BD%D0%B4%D0%B0" title="Руанда">Руанда</a>, <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%B7%D0%B0%D0%BD%D0%B8%D1%8F" title="Танзания">Танзания</a> и <a href="/wiki/%D0%97%D0%B0%D0%BC%D0%B1%D0%B8%D1%8F" title="Замбия">Замбия</a>. Площадь его непосредственного <a href="/wiki/%D0%90%D1%80%D0%B5%D0%B0%D0%BB" title="Ареал">ареала</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">extend of occurence</span>) составляет 3 780 000<span class="nowrap"> км²</span><sup class="reference" id="cite_ref-iucn_7-0"><a href="#cite_note-iucn-7"><span class="cite-bracket">[</span>7<span class="cite-bracket">]</span></a></sup>. Ареал вида лоскутный<sup class="reference" id="cite_ref-_b96edb379d04b541_6-1"><a href="#cite_note-_b96edb379d04b541-6"><span class="cite-bracket">[</span>6<span class="cite-bracket">]</span></a></sup>.
</p><p>Подвид <i>S. l. lugens</i> обитает близ города <a href="/wiki/%D0%9E%D0%B1%D0%B0%D0%BB%D0%B0" title="Обала">Обала</a> в Камеруне, на северо-востоке Габона, в районах <span data-interwiki-article="Faradje" data-interwiki-lang="fr"><a class="new" href="/w/index.php?title=%D0%A4%D0%B0%D1%80%D0%B0%D0%B4%D0%B6%D0%B5&amp;action=edit&amp;redlink=1" title="Фарадже (страница отсутствует)">Фарадже</a></span><sup class="noprint" style="font-style:normal; font-weight:normal;"><a class="extiw" href="https://fr.wikipedia.org/wiki/Faradje" title="fr:Faradje"><span title="Faradje — версия статьи «Фарадже» на французском языке">[фр.]</span></a></sup>, Касажи (<i>Kasaji</i>), Кунунгу (<i>Kunungu</i>) Демократической Республики Конго, в Руанде и в районе реки Угалла (<i>Ugalla River</i>) на западе Танзании. Подвид <i>S. l. lynesi</i> обитает в провинции <a href="/wiki/%D0%91%D0%B8%D0%B5" title="Бие">Бие</a> в центральной части Анголы и на севере Замбии. Возможно, вид встречается в <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%B0%D0%B2%D0%B8" title="Малави">Малави</a>, но в этом регионе широко распространён красногрудый пушистый погоныш и как минимум одна из двух записей позывок относится к этому виду<sup class="reference" id="cite_ref-taylor_3-6"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>. Учёными были отклонены наблюдения каштановоголового пушистого погоныша в <a href="/wiki/%D0%97%D0%B8%D0%BC%D0%B1%D0%B0%D0%B1%D0%B2%D0%B5" title="Зимбабве">Зимбабве</a>, <a href="/wiki/%D0%9D%D0%B8%D0%B3%D0%B5%D1%80%D0%B8%D1%8F" title="Нигерия">Нигерии</a>, <a href="/wiki/%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0_%D0%9A%D0%BE%D0%BD%D0%B3%D0%BE" title="Республика Конго">Республике Конго</a> и <a href="/wiki/%D0%A6%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B0%D1%84%D1%80%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0" title="Центральноафриканская Республика">Центральноафриканской Республике</a><sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-5"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-taylor_3-7"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>. <span data-interwiki-article="Stuart Keith" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9A%D0%B8%D1%82,_%D0%A1%D1%82%D1%8E%D0%B0%D1%80%D1%82&amp;action=edit&amp;redlink=1" title="Кит, Стюарт (страница отсутствует)">Стюарт Кит</a></span><sup class="noprint" style="font-style:normal; font-weight:normal;"><a class="extiw" href="https://en.wikipedia.org/wiki/Stuart_Keith" title="en:Stuart Keith"><span title="Stuart Keith — версия статьи «Кит, Стюарт» на английском языке">[англ.]</span></a></sup>, <a href="/wiki/%D0%91%D0%B5%D0%BD%D1%81%D0%BE%D0%BD,_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_%D0%A3%D0%BE%D0%BB%D1%82%D0%B5%D1%80" title="Бенсон, Константин Уолтер">Константин Уолтер Бенсон</a>, <span data-interwiki-article="Michael Patrick Stuart Irwin" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%98%D1%80%D0%B2%D0%B8%D0%BD,_%D0%9C%D0%B0%D0%B9%D0%BA%D0%BB_%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%BA_%D0%A1%D1%82%D1%8E%D0%B0%D1%80%D1%82&amp;action=edit&amp;redlink=1" title="Ирвин, Майкл Патрик Стюарт (страница отсутствует)">Майкл Патрик Стюарт Ирвин</a></span><sup class="noprint" style="font-style:normal; font-weight:normal;"><a class="extiw" href="https://en.wikipedia.org/wiki/Michael_Patrick_Stuart_Irwin" title="en:Michael Patrick Stuart Irwin"><span title="Michael Patrick Stuart Irwin — версия статьи «Ирвин, Майкл Патрик Стюарт» на английском языке">[англ.]</span></a></sup> в работе 1970 года задавались вопросом, почему обитающий близ <a href="/wiki/%D0%AD%D0%BA%D0%B2%D0%B0%D1%82%D0%BE%D1%80" title="Экватор">экватора</a> номинативный подвид <i>S. l. lugens</i> крупнее <i>S. l. lynesi</i>. Такое распространение противоречит <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%B8%D0%BB%D0%BE_%D0%91%D0%B5%D1%80%D0%B3%D0%BC%D0%B0%D0%BD%D0%B0" title="Правило Бергмана">правилу Бергмана</a><sup class="reference" id="cite_ref-_d2302afc0ad7fbea_4-1"><a href="#cite_note-_d2302afc0ad7fbea-4"><span class="cite-bracket">[</span>4<span class="cite-bracket">]</span></a></sup>.
</p><p>Каштановоголовый пушистый погоныш обитает на небольших участках <a href="/wiki/%D0%A1%D0%B0%D0%B2%D0%B0%D0%BD%D0%BD%D0%B0" title="Саванна">саванны</a> в равнинных лесах или на покрытых травой болотах, на болотистых берегах озёр, в высокой траве по краю вудленда — «дамбо» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">dambo</span>). Часто встречается в местах с густой растительностью высотой <span class="nowrap">0,7—1,5 м</span>, но не любит глубокие водоёмы или грязь. На влажных культивируемых ландшафтах на северо-востоке Габона с доминирующими растениями рода <a href="/wiki/%D0%90%D1%84%D1%80%D0%B0%D0%BC%D0%BE%D0%BC%D1%83%D0%BC" title="Афрамомум">афрамомум</a> (<i>Aframomum</i>) и <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B0%D0%BD%D1%82%D0%B0_%D1%82%D1%80%D0%BE%D1%81%D1%82%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2%D0%B8%D0%B4%D0%BD%D0%B0%D1%8F" title="Маранта тростниковидная">марантой тростниковидной</a> (<i>Maranta arundinacea</i>) каштановоголовый пушистый погоныш встречается вместе с красногрудым, хотя, как и другие пушистые погоныши, предпочитает более сухие территории. Аналогично, на севере Замбии оба вида встречаются в высокой траве и делят ареал с африканским пушистым погонышем, который, впрочем, предпочитает менее высокую растительность<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-6"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-taylor_3-8"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>.
</p><p>Скорее всего, каштановоголовый пушистый погоныш ведёт полностью <a href="/wiki/%D0%9C%D0%B8%D0%B3%D1%80%D0%B0%D1%86%D0%B8%D1%8F_%D0%BF%D1%82%D0%B8%D1%86" title="Миграция птиц">оседлый образ жизни</a>, осуществляя локальные перемещения за пределами сезона размножения из-за потери среды обитания вследствие пожаров<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-7"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-iucn_7-1"><a href="#cite_note-iucn-7"><span class="cite-bracket">[</span>7<span class="cite-bracket">]</span></a></sup>. Возможно, после сезона размножения вид разлетается довольно широко, так как, вероятно, один раз птица была отмечена в Малави<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-8"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-taylor_3-9"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>.
</p>
<div class="mw-heading mw-heading3"><h3 id="Охранный_статус"><span id=".D0.9E.D1.85.D1.80.D0.B0.D0.BD.D0.BD.D1.8B.D0.B9_.D1.81.D1.82.D0.B0.D1.82.D1.83.D1.81"></span>Охранный статус</h3><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Охранный статус»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;action=edit&amp;section=4" title="Редактировать код раздела «Охранный статус»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7_%D0%BE%D1%85%D1%80%D0%B0%D0%BD%D1%8B_%D0%BF%D1%80%D0%B8%D1%80%D0%BE%D0%B4%D1%8B" title="Международный союз охраны природы">Международный союз охраны природы</a> относит каштановоголового пушистого погоныша к <a href="/wiki/%D0%92%D0%B8%D0%B4%D1%8B,_%D0%B1%D0%BB%D0%B8%D0%B7%D0%BA%D0%B8%D0%B5_%D0%BA_%D1%83%D1%8F%D0%B7%D0%B2%D0%B8%D0%BC%D0%BE%D0%BC%D1%83_%D0%BF%D0%BE%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D1%8E" title="Виды, близкие к уязвимому положению">видам, близким к уязвимому положению</a> (<i>NT</i>)<sup class="reference" id="cite_ref-iucn_7-2"><a href="#cite_note-iucn-7"><span class="cite-bracket">[</span>7<span class="cite-bracket">]</span></a></sup>. Кит, Бенсон и Ирвин полагали, что это <a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D0%BB%D0%B8%D0%BA%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%B8%D0%B4" title="Реликтовый вид">реликтовый вид</a>, который уступает в распространении родственному красногрудому пушистому погонышу<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-9"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-_2e36a133df8b92c3_8-0"><a href="#cite_note-_2e36a133df8b92c3-8"><span class="cite-bracket">[</span>8<span class="cite-bracket">]</span></a></sup>. Вместе с тем, на некоторых участках его численность выше, чем у последнего. Птицы распространены на северо-востоке Замбии, где 5-6 поющих птиц отмечали на участке радиусом около 450 м. Оценка затруднена из-за некорректной идентификации молодых самок<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-10"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup>. Численность оценивается в 670-6700 взрослых особей и, предположительно, падает<sup class="reference" id="cite_ref-iucn_7-3"><a href="#cite_note-iucn-7"><span class="cite-bracket">[</span>7<span class="cite-bracket">]</span></a></sup>. По мнению автора книги о пастушковых Барри Тейлора (<i>Barry Taylor</i>), каштановоголовый пушистый погоныш должен быть отнесён к <a href="/wiki/%D0%A3%D1%8F%D0%B7%D0%B2%D0%B8%D0%BC%D1%8B%D0%B5_%D0%B2%D0%B8%D0%B4%D1%8B" title="Уязвимые виды">уязвимым видам</a> (<i>VU</i>)<sup class="reference" id="cite_ref-taylor_3-10"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>.
</p>
<div class="mw-heading mw-heading2"><h2 id="Питание"><span id=".D0.9F.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D0.B5"></span>Питание</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Питание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;action=edit&amp;section=5" title="Редактировать код раздела «Питание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Очень мало известно об особенностях питания каштановоголового пушистого погоныша. В желудках взрослых птиц были обнаружены <a href="/wiki/%D0%9D%D0%B0%D1%81%D0%B5%D0%BA%D0%BE%D0%BC%D1%8B%D0%B5" title="Насекомые">насекомые</a> и семена, в желудке птенца — маленькие чёрные <a href="/wiki/%D0%9C%D1%83%D1%80%D0%B0%D0%B2%D1%8C%D0%B8" title="Муравьи">муравьи</a><sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-11"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-_b96edb379d04b541_6-2"><a href="#cite_note-_b96edb379d04b541-6"><span class="cite-bracket">[</span>6<span class="cite-bracket">]</span></a></sup>.
</p>
<div class="mw-heading mw-heading2"><h2 id="Размножение"><span id=".D0.A0.D0.B0.D0.B7.D0.BC.D0.BD.D0.BE.D0.B6.D0.B5.D0.BD.D0.B8.D0.B5"></span>Размножение</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Размножение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;action=edit&amp;section=6" title="Редактировать код раздела «Размножение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Сезон размножения варьирует в зависимости от географического положения; в южной части ареала приходится на время влажного сезона<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-12"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-iucn_7-4"><a href="#cite_note-iucn-7"><span class="cite-bracket">[</span>7<span class="cite-bracket">]</span></a></sup>. Предположительно, птицы остаются одиночками в течение всего года, формируя пары только на время размножения<sup class="reference" id="cite_ref-iucn_7-5"><a href="#cite_note-iucn-7"><span class="cite-bracket">[</span>7<span class="cite-bracket">]</span></a></sup>.
</p><p>На основе информации о молодых птицах и о самках, готовых откладывать яйца, учёные выяснили, что каштановоголовый пушистый погоныш <a href="/wiki/%D0%9A%D0%BB%D0%B0%D0%B4%D0%BA%D0%B0_%D1%8F%D0%B8%D1%86" title="Кладка яиц">откладывает яйца</a> в Камеруне в апреле, июле и сентябре, в Демократической Республике Конго — в марте и апреле, в Замбии — в марте и декабре. <a href="/wiki/%D0%93%D0%BD%D0%B5%D0%B7%D0%B4%D0%BE_%D0%BF%D1%82%D0%B8%D1%86%D1%8B" title="Гнездо птицы">Гнёзда</a> и <a href="/wiki/%D0%AF%D0%B9%D1%86%D0%BE" title="Яйцо">яйца</a> этого вида описаны не были<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-13"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-taylor_3-11"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup>.
</p><p>Единственный известный экземпляр птенца был покрыт чёрным пухом, его клюв окрашен в чёрный цвет с белым основанием и кончиком<sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-14"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup>.
</p>

<p>Каштановоголовый пушистый погоныш был описан в 1884 году немецким зоологом <a href="/wiki/%D0%91%D1%91%D0%BC,_%D0%A0%D0%B8%D1%85%D0%B0%D1%80%D0%B4" title="Бём, Рихард">Рихардом Бёмом</a>, который отнёс его к роду <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D1%81%D1%82%D0%B5%D0%BB%D0%B8" title="Коростели">коростели</a> (<i>Crex</i>) — <i>Crex luhens</i><sup class="reference" id="cite_ref-taylor_3-12"><a href="#cite_note-taylor-3"><span class="cite-bracket">[</span>3<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-_b96edb379d04b541_6-3"><a href="#cite_note-_b96edb379d04b541-6"><span class="cite-bracket">[</span>6<span class="cite-bracket">]</span></a></sup>. В 1934 году британский орнитолог и адмирал флота <span data-interwiki-article="Hubert Lynes" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9B%D0%B8%D0%BD%D1%81,_%D0%A5%D1%8C%D1%8E%D0%B1%D0%B5%D1%80%D1%82&amp;action=edit&amp;redlink=1" title="Линс, Хьюберт (страница отсутствует)">Хьюберт Линс</a></span><sup class="noprint" style="font-style:normal; font-weight:normal;"><a class="extiw" href="https://en.wikipedia.org/wiki/Hubert_Lynes" title="en:Hubert Lynes"><span title="Hubert Lynes — версия статьи «Линс, Хьюберт» на английском языке">[англ.]</span></a></sup> обнаружил два экземпляра птиц, которых он отнёс к <i>Sarothrura rufa</i>. <span data-interwiki-article="Claude Henry Baxter Grant" data-interwiki-lang="de"><a class="new" href="/w/index.php?title=%D0%93%D1%80%D0%B0%D0%BD%D1%82,_%D0%9A%D0%BB%D0%BE%D0%B4_%D0%93%D0%B5%D0%BD%D1%80%D0%B8_%D0%91%D0%B0%D0%BA%D1%81%D1%82%D0%B5%D1%80&amp;action=edit&amp;redlink=1" title="Грант, Клод Генри Бакстер (страница отсутствует)">Клод Генри Бакстер Грант</a></span><sup class="noprint" style="font-style:normal; font-weight:normal;"><a class="extiw" href="https://de.wikipedia.org/wiki/Claude_Henry_Baxter_Grant" title="de:Claude Henry Baxter Grant"><span title="Claude Henry Baxter Grant — версия статьи «Грант, Клод Генри Бакстер» на немецком языке">[нем.]</span></a></sup> и <span data-interwiki-article="Cyril Mackworth-Praed" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9C%D0%B0%D0%BA%D0%B2%D0%BE%D1%80%D1%82-%D0%9F%D1%80%D0%B0%D0%B5%D0%B4,_%D0%A1%D0%B8%D1%80%D0%B8%D0%BB&amp;action=edit&amp;redlink=1" title="Макворт-Праед, Сирил (страница отсутствует)">Сирил Макворт-Праед</a></span><sup class="noprint" style="font-style:normal; font-weight:normal;"><a class="extiw" href="https://en.wikipedia.org/wiki/Cyril_Mackworth-Praed" title="en:Cyril Mackworth-Praed"><span title="Cyril Mackworth-Praed — версия статьи «Макворт-Праед, Сирил» на английском языке">[англ.]</span></a></sup> в 1934 году описали их как <i>Sarothrura lineata lynesi</i> — подвид <a href="/wiki/%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D1%85%D0%B2%D0%BE%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Каштановохвостый пушистый погоныш">каштановохвостого пушистого погоныша</a> (<i>Sarothrura affinis</i>), а с 1937 года рассматривали его как отдельный вид<sup class="reference" id="cite_ref-_b96edb379d04b541_6-4"><a href="#cite_note-_b96edb379d04b541-6"><span class="cite-bracket">[</span>6<span class="cite-bracket">]</span></a></sup>. С 1965 года многие учёные стали рассматривать <i>Sarothrura lynesi</i> и <i>Sarothrura lugens</i> конспецифическими<sup class="reference" id="cite_ref-roux_5-1"><a href="#cite_note-roux-5"><span class="cite-bracket">[</span>5<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-_d2302afc0ad7fbea_4-2"><a href="#cite_note-_d2302afc0ad7fbea-4"><span class="cite-bracket">[</span>4<span class="cite-bracket">]</span></a></sup>. В 1949 году швейцарский зоолог Альбер Монар (<i>Albert Monard</i>) описал вид <i>Sarothrura modesta</i><sup class="reference" id="cite_ref-monard_9-0"><a href="#cite_note-monard-9"><span class="cite-bracket">[</span>9<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-roux_5-2"><a href="#cite_note-roux-5"><span class="cite-bracket">[</span>5<span class="cite-bracket">]</span></a></sup>, который в 1951 году был признан синонимом <i>Sarothrura lugens</i><sup class="reference" id="cite_ref-_b96edb379d04b541_6-5"><a href="#cite_note-_b96edb379d04b541-6"><span class="cite-bracket">[</span>6<span class="cite-bracket">]</span></a></sup>.
</p><p><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7_%D0%BE%D1%80%D0%BD%D0%B8%D1%82%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B2" title="Международный союз орнитологов">Международный союз орнитологов</a> выделяет два подвида<sup class="reference" id="cite_ref-ioc_10-0"><a href="#cite_note-ioc-10"><span class="cite-bracket">[</span>10<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-bow_sarothrura_lugens_2-15"><a href="#cite_note-bow_sarothrura_lugens-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup>:
</p>
<ul><li><span data-wikidata-property-id="P225" lang="la" style="font-style: italic;">Sarothrura lugens lugens</span> <span style="font-variant:small-caps;">(Boem, 1884)</span> — обитает от Камеруна до Демократической Республики Конго, в Руанде и на западе Танзании;</li>
<li><span data-wikidata-property-id="P225" lang="la" style="font-style: italic;">Sarothrura lugens lynesi</span> <span style="font-variant:small-caps;">Grant, Mackworth-Praed, 1934</span> — обитает в центральной Анголе, Замбии и, возможно, Малави.</li></ul>

href="/wiki/%D0%91%D0%B0%D0%B7%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0" title="Базальная группа">базальных</a> <a погоныш">жемчужного</a> (<i>Sarothrura pulchra</i>) и <a href="/wiki/%D0%9F%D1%91%D1%81%D1%82%D1%80%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Пёстрый пушистый погоныш">пёстрого</a> (<i>Sarothrura elegans</i>) пушистых погонышей, «ayresi» — <a class="new" href="/w/index.php?title=%D0%97%D0%B5%D1%80%D0%BA%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;action=edit&amp;redlink=1" title="Зеркальный пушистый погоныш (страница отсутствует)">зеркального пушистого погоныша</a> (<i>Sarothrura ayresi</i>) и <a href="/wiki/%D0%9B%D0%B5%D0%BC%D1%83%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Лемурийский погоныш">лемурийского погоныша</a> (<i>Sarothrura watersi</i>), «affinis» — <a href="/wiki/%D0%9A%D0%B0%D1%88%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE%D1%85%D0%B2%D0%BE%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Каштановохвостый пушистый погоныш">каштановохвостого</a> (<i>Sarothrura affinis</i>) и <a class="new" href="/w/index.php?title=%D0%9C%D0%B0%D0%B4%D0%B0%D0%B3%D0%B0%D1%81%D0%BA%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88&amp;action=edit&amp;redlink=1" title="Мадагаскарский пушистый погоныш (страница отсутствует)">мадагаскарского</a> (<i>Sarothrura insularis</i>) пушистых погонышей, «rufa» — <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D0%BE%D0%B3%D1%80%D1%83%D0%B4%D1%8B%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Красногрудый пушистый погоныш">красногрудого</a> (<i>Sarothrura rufa</i>), <a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88" title="Африканский пушистый погоныш">африканского</a> (<i>Sarothrura boehmi</i>) и каштановоголового пушистых погонышей<sup class="reference" id="cite_ref-_b96edb379d04b541_6-6"><a href="#cite_note-_b96edb379d04b541-6"><span class="cite-bracket">[</span>6<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-_f41189c325e6f2c0_11-0"><a href="#cite_note-_f41189c325e6f2c0-11"><span class="cite-bracket">[</span>11<span class="cite-bracket">]</span></a></sup>. В своей работе по классификации <a href="/wiki/%D0%9F%D0%B0%D1%81%D1%82%D1%83%D1%88%D0%BA%D0%BE%D0%B2%D1%8B%D0%B5" title="Пастушковые">пастушковых</a> (Rallidae), опубликованной в 1973 году, <a href="/wiki/%D0%9E%D0%BB%D1%81%D0%BE%D0%BD,_%D0%A1%D1%82%D0%BE%D1%80%D1%80%D1%81_%D0%9B%D0%B0%D0%B2%D0%B4%D0%B6%D0%BE%D0%B9" title="Олсон, Сторрс Лавджой">Сторрс Лавджой Олсон</a> также считал обитающего в лесах жемчужного пушистого погоныша базальным таксоном, но утверждал, что остальные постепенно адаптировались к жизни в полях и всё больше отличались от <i><a class="mw-redirect" href="/wiki/Rallicula" title="Rallicula">Rallicula</a></i><sup class="reference" id="cite_ref-_4738b77ecc9ae869_12-0"><a href="#cite_note-_4738b77ecc9ae869-12"><span class="cite-bracket">[</span>12<span class="cite-bracket">]</span></a></sup>. Вместе с тем, <span data-interwiki-article="Bradley C. Livezey" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9B%D0%B8%D0%B2%D0%B7%D0%B8,_%D0%91%D1%80%D0%B5%D0%B4%D0%BB%D0%B8&amp;action=edit&amp;redlink=1" title="Ливзи, Бредли (страница отсутствует)">Бредли Ливзи</a></span><sup class="noprint" style="font-style:normal; font-weight:normal;"><a class="extiw" href="https://en.wikipedia.org/wiki/Bradley_C._Livezey" title="en:Bradley C. Livezey"><span title="Bradley C. Livezey — версия статьи «Ливзи, Бредли» на английском языке">[англ.]</span></a></sup> назвал разделение Кита, Бенсона и Ирвина интуитивным, но очень близким к результатам его исследований, опубликованным в 1998 году<sup class="reference" id="cite_ref-_f41189c325e6f2c0_11-1"><a href="#cite_note-_f41189c325e6f2c0-11"><span class="cite-bracket">[</span>11<span class="cite-bracket">]</span></a></sup>. По его мнению, жемчужный, пёстрый и красногрудый пушистые погоныши являются базальными таксонами, <a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D1%80%D0%B0%D1%84%D0%B8%D0%BB%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0%D1%82%D0%B8%D0%BA%D0%B0)" title="Парафилия (биологическая систематика)">парафилитическими</a> по отношению к остальным шести видам, среди которых <a href="/wiki/%D0%A1%D0%B5%D1%81%D1%82%D1%80%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0" title="Сестринская группа">сестринские</a> отношения демонстрируют зеркальный и лемурийский, каштановоголовый и африканский, мадагаскарский и каштановохвостый виды. Последние две группы сестринские по отношению к друг другу, а первая — по отношению к образовавшейся группе<sup class="reference" id="cite_ref-_f40e1cc325e403d6_13-0"><a href="#cite_note-_f40e1cc325e403d6-13"><span class="cite-bracket">[</span>13<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-_f40e1cc325e403d0_14-0"><a href="#cite_note-_f40e1cc325e403d0-14"><span class="cite-bracket">[</span>14<span class="cite-bracket">]</span></a></sup>.
</p><p>Род <a href="/wiki/%D0%9F%D1%83%D1%88%D0%B8%D1%81%D1%82%D1%8B%D0%B5_%D0%BF%D0%BE%D0%B3%D0%BE%D0%BD%D1%8B%D1%88%D0%B8" title="Пушистые погоныши">пушистые погоныши</a> (<i>Sarothrura</i>) формирует основу семейства <a href="/wiki/Sarothruridae" title="Sarothruridae">Sarothruridae</a>, выделенного в XXI веке из семейства пастушковых<sup class="reference" id="cite_ref-bow_sarothruridae_15-0"><a href="#cite_note-bow_sarothruridae-15"><span class="cite-bracket">[</span>15<span class="cite-bracket">]</span></a></sup><sup class="reference" id="cite_ref-boast_16-0"><a href="#cite_note-boast-16"><span class="cite-bracket">[</span>16<span class="cite-bracket">]</span></a></sup>. Помимо пушистых погонышей Международный союз орнитологов относит к этому семейству представителей родов <i><a href="/wiki/Mentocrex" title="Mentocrex">Mentocrex</a></i> и <i>Rallicula</i><sup class="reference" id="cite_ref-ioc_10-1"><a href="#cite_note-ioc-10"><span class="cite-bracket">[</span>10<span class="cite-bracket">]</span></a></sup>.
</p>
